Welcome to a sneak-preview of the new Campus Cloud!


Our team is hard at work on a new, improved version of Campus Cloud. We hope to get valuable feedback by allowing certain customers such as yourself to use it at this early stage.


This early preview may not have all the features you expect (yet!), but we're adding new features regularly.

New as of July 10, 2017:

  • Administrators can send Notifications to students' devices
  • Administrators can create lists of users, to use in sending Notifications.
  • Bug fixes and usability improvements

New as of June 2, 2017:

  • Administrators can create Clubs and Groups
  • Executives of Clubs and Groups can create Events and Services within those groups
  • Students can post messages to Club Feeds. Executives and Administrators can moderate these Feeds.
